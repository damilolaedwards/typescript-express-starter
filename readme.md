### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/tipshot-net/encrypt-decrypt-node.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Create a `config` folder in the root directory.
   copy the contents of `config.example` to `config` & set env variables

4. Run :rocket:

   ```sh
   npm run dev
   ```
